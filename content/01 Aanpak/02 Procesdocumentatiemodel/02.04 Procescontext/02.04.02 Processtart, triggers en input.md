Deze sectie beschrijft hoe een proces wordt gestart, welke gebeurtenissen dit veroorzaken en welke input nodig is om het proces uit te voeren.
#### Processtart

De processtart is het moment waarop een proces officieel begint. Dit kan:

- automatisch gebeuren (systeemtrigger)  
- handmatig gebeuren (menselijke actie)  
- extern worden geïnitieerd (klant of partner)  
#### Triggers

Triggers zijn gebeurtenissen die het proces activeren.

Voorbeelden:

- klant plaatst bestelling  
- systeem ontvangt EDI-bericht  
- medewerker registreert aanvraag  
- voorraadniveau bereikt drempelwaarde  
#### Input

Input bestaat uit alle gegevens en informatie die nodig zijn om het proces uit te voeren.

##### Typische input

- klantgegevens  
- productinformatie  
- orderspecificaties  
- prijsafspraken  
- systeemdata  

##### Kwaliteit van input

Goede procesuitvoering vereist dat input:

- volledig is  
- correct is  
- actueel is  
- beschikbaar is op het juiste moment  

Onvolledige input is een van de belangrijkste oorzaken van procesfouten.
