Deze sectie beschrijft hoe processen binnen het PDM met elkaar verbonden zijn.

#### Doel

Relaties maken inzichtelijk:

- afhankelijkheden tussen processen  
- procesketens en end-to-end flows  
- overdrachtsmomenten tussen processen  
- impact van wijzigingen op andere processen  
#### Soorten relaties

| Relatie                | Omschrijving                               |
| ---------------------- | ------------------------------------------ |
| Input-output relaties  | Output van proces A is input voor proces B |
| Sequentiële relaties   | Proces volgt logisch op een ander proces   |
| Parallelle relaties    | Processen lopen gelijktijdig naast elkaar  |
| Hiërarchische relaties | Subproces binnen hoofdproces               |
#### Voorbeeld (Orderverwerking)

- Orderverwerking → Facturatie  
- Orderverwerking → Levering  
- Orderverwerking ← Orderintake  

#### Belangrijk principe

Geen enkel proces staat volledig op zichzelf binnen het PDM.
