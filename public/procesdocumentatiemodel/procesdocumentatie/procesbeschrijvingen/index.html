Procesbeschrijvingen

Een procesdiagram laat zien hoe een proces verloopt. Een procesbeschrijving geeft meer detail over de activiteiten binnen het proces.

Een goede procesbeschrijving bevat onder andere:

  • het doel van het proces
  • de belangrijkste processtappen
  • input en output
  • gebruikte systemen
  • uitzonderingen of bijzonderheden

Structuur van een procesbeschrijving

Een eenvoudige structuur kan er als volgt uitzien:

Onderdeel Beschrijving
Procesnaam Naam van het proces
Doel Waarom het proces bestaat
Input Wat nodig is om het proces te starten
Activiteiten De stappen binnen het proces
Output Het resultaat van het proces
Rollen Wie betrokken zijn

Voorbeeld

Proces: Orderverwerking

Doel:
Het correct verwerken van klantorders en het leveren van producten aan de klant.

Belangrijkste stappen:

  1. Order ontvangen
  2. Order controleren
  3. Order picken
  4. Order verzenden
  5. Factureren

Door processen op deze manier te beschrijven ontstaat een duidelijke werkinstructie voor medewerkers.
